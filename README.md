Goal

To make an Android app that lets the user choose their gender with radio buttons and shows the choice when a button is clicked.

Algorithm 
Start the program.
There should be three choices: Male, Female, and Other.
Give the user time to choose one option.
When the user clicks the button that says "Show Selected":
Make sure that one of the options is checked.
If nothing is chosen, show a message that says to pick an option.
If you choose a gender, show it.
The end.
